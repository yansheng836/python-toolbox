# 工具箱 — 用户手册

欢迎使用**工具箱**！这是一款离线、高效、美观的桌面工具集合，帮您轻松处理图片和 PDF 文件。

---

## 快速开始

1. 双击打开工具箱程序
2. 您会看到左侧是工具菜单，右侧是内容区域
3. 点击左侧任意工具名称，即可开始使用
4. 右侧首页会展示所有可用功能，您也可以从这里快速进入

> 💡 **小贴士**：把鼠标移到左侧菜单上，可以看到每个工具的图标和名称。

---

## 界面介绍

工具箱的界面分为两大部分：

### 左侧：工具菜单

- 列出了所有可用工具
- 点击工具名称即可切换到对应功能
- 点击侧边栏右边缘的箭头按钮可折叠/展开菜单，折叠后只显示图标，节省工作区域空间
- 底部有「设置」入口，可以切换主题或查看关于信息

### 右侧：工作区域

- **首页**：展示所有功能特色，点击卡片可快速进入
- **工具页**：每个工具的操作界面，包含文件列表、操作按钮、进度显示等
- **设置页**：切换深色/浅色主题，查看软件信息

### 通用操作元素

| 元素 | 说明 |
|------|------|
| 📄 添加文件 | 点击后选择文件添加到列表 |
| 🗑️ 移除选中 | 删除列表中选中的文件 |
| 🧹 清空列表 | 清空整个文件列表 |
| ⬆ 上移 / ⬇ 下移 | 调整文件在列表中的顺序（部分工具支持） |
| 🔼 开始处理 | 启动当前工具的任务 |
| 进度条 | 显示任务处理进度 |
| 状态提示 | 显示任务状态和结果信息 |

## 功能详解

### 图片压缩

![图片压缩](README.assets/图片压缩.png)

**功能说明**：批量压缩 JPG、PNG、WebP 格式的图片，减小文件体积，方便分享和存储。

**使用步骤**：

1. 点击左侧「图片压缩」菜单
2. 点击「添加图片」按钮，选择要压缩的图片（可多选）
3. 在「输出格式」下拉菜单中选择格式：
   - **保持原格式**：压缩后格式不变
   - **JPG / JPEG**：转为 JPG 格式
   - **PNG**：转为 PNG 格式
   - **WebP**：转为 WebP 格式（更小的体积）
4. 拖动「压缩质量」滑块调整质量（1~100%）：
   - 数值越高，画质越好，文件越大
   - 建议设置为 70~85，兼顾画质和体积
5. 选择输出目录（可选，不选择则保存到原图目录）
6. 点击「开始压缩」按钮
7. 等待处理完成，查看结果提示

> 📌 **适用场景**：邮件附件太大发不出去？微信传图被压缩？用这个功能批量压缩后再分享！

### 图片格式转换

![图片格式转换](README.assets/图片格式转换.png)

**功能说明**：批量将图片转换为另一种格式，支持 JPEG、PNG、WebP、BMP、TIFF、GIF 六种格式。

**使用步骤**：

1. 点击左侧「图片格式转换」菜单
2. 点击「添加图片」按钮，选择要转换的图片（可多选）
3. 在「目标格式」下拉菜单中选择想要的格式
4. 设置「转换质量」滑块（1~100%）
5. 选择输出目录（可选）
6. 点击「开始转换」按钮
7. 等待处理完成

> 📌 **适用场景**：网站要求上传 PNG 格式但您只有 JPG？批量转换，一键搞定！

### 图片拼接

![图片拼接](README.assets/图片拼接.png)

**功能说明**：将多张图片拼接成一张大图，支持横向拼接（左右排列）或纵向拼接（上下排列）。

**使用步骤**：

1. 点击左侧「图片拼接」菜单
2. 点击「添加图片」按钮，选择要拼接的图片（至少 2 张）
3. 调整图片顺序（使用「上移」「下移」按钮）
4. 选择拼接方向：
   - **横向拼接**：图片从左到右排列
   - **纵向拼接**：图片从上到下排列
5. 设置「图片质量」滑块（1~100%）
6. 点击「浏览」选择输出文件路径
7. 点击「开始拼接」按钮
8. 等待处理完成

> 📌 **适用场景**：把多张截图拼成长图方便分享？把多张照片拼成一张做对比？这个功能最适合！

### 图片转PDF

![图片转PDF](README.assets/图片转PDF.png)

**功能说明**：将多张图片合并成一个 PDF 文件，支持拖拽调整顺序。

**使用步骤**：

1. 点击左侧「图片转PDF」菜单
2. 添加图片，有以下方式：
   - 点击「添加图片」按钮选择文件
   - 直接将图片文件拖拽到文件列表区域
3. 调整图片顺序（使用「上移」「下移」按钮或「按名称排序」）
4. 选择「输出格式」：
   - **高质量（PDF/A）**：文件较大，画质最佳
   - **标准质量**：平衡画质和体积
   - **压缩质量**：文件最小，适合分享
5. 点击「浏览」选择输出 PDF 文件路径
6. 点击「开始转换」按钮
7. 等待处理完成

> 📌 **适用场景**：把多张照片打包成一个 PDF 发给客户？把扫描件合并成一份文档？轻松搞定！

### 图片缩放

![图片缩放](README.assets/图片缩放.png)

**功能说明**：批量调整图片尺寸，支持按比例缩放或指定具体宽度/高度。

**使用步骤**：

1. 点击左侧「图片缩放」菜单
2. 点击「添加图片」按钮，选择要缩放的图片（可多选）
3. 选择缩放方式：
   - **按比例**：输入百分比（如 50% 表示缩小一半）
   - **按宽度**：指定目标宽度，高度自动按比例调整
   - **按高度**：指定目标高度，宽度自动按比例调整
   - **指定宽高**：输入具体宽度和高度数值
4. 勾选「保持宽高比」可防止图片变形（建议勾选）
5. 设置「输出质量」滑块（1~100%）
6. 选择输出目录（可选）
7. 点击「开始缩放」按钮
8. 等待处理完成

> 📌 **适用场景**：网站要求图片宽度不超过 800 像素？批量调整几百张图片尺寸？一键搞定！

![PDF合并](README.assets/PDF合并.png)

**功能说明**：将多个 PDF 文件合并成一个，支持拖拽调整顺序。

**使用步骤**：

5. 点击「开始合并」按钮

### PDF拆分

![PDF拆分](README.assets/PDF拆分.png)

**功能说明**：将一个 PDF 文件拆分成多个小文件，可以按固定页数拆分，也可以将每一页单独导出为图片。

**使用步骤**：

1. 点击左侧「PDF拆分」菜单
2. 点击「添加PDF」按钮，选择一个 PDF 文件
3. 选择输出格式：
   - **单页PDF**：每 N 页生成一个 PDF 文件
   - **图片**：将每一页导出为图片（JPG/PNG/WebP）
4. 如果选择「单页PDF」：
   - 设置「拆分页数」（如设置为 1，则每页生成一个 PDF；设置为 5，则每 5 页生成一个 PDF）
5. 如果选择「图片」：
   - 选择图片格式（JPG/PNG/WebP）
   - 设置图片质量滑块
6. 选择输出目录（默认保存到原文件所在目录）
7. 点击「开始拆分」按钮
8. 等待处理完成

> 📌 **适用场景**：把一份 100 页的 PDF 拆成单页方便分享？把 PDF 转成图片插入文档？这个功能帮您！

### 文件去重

![文件去重](README.assets/文件去重.png)

**功能说明**：扫描文件夹，找出内容完全相同的重复文件（通过文件内容计算，不是文件名），帮您清理重复文件节省空间。采用三级筛选策略（文件大小→快速Hash→完整Hash），大幅提速扫描过程。

**使用步骤**：

1. 点击左侧「文件去重」菜单
2. 点击「选择文件夹」按钮，选择要扫描的文件夹，自动开始扫描
3. 等待扫描完成，结果会按重复组展示在树形列表中
4. 选择删除规则：
   - **按文件名升序排序**（A→Z，保留首个，删除后续）
   - **按文件名降序排序**（Z→A，保留首个，删除后续）
   - **智能识别副本**（优先删除含 (1)/_copy/_backup 等标记的文件，其余按创建时间排序）
5. 点击「按规则排序」按钮，预览每组文件的保留/删除标记（✅ 保留 / ❌ 删除）
6. 勾选要处理的重复文件组
7. 点击「删除重复文件」按钮
8. 在确认对话框中核对删除列表，点击「确认」完成删除

> 💡 **小贴士**：勾选「删除后自动扫描」可在删除完成后自动重新扫描，方便连续清理。

> ⚠️ **注意**：删除文件前请仔细确认！建议先备份重要文件。去重操作不可撤销。只读文件也会被自动解除权限后删除。

> 📌 **适用场景**：手机传照片到电脑后有很多重复？下载的文件散落各处有重复？扫描一下，轻松找出所有重复文件！

## 设置与主题

### 切换主题

![设置](README.assets/设置.png)

工具箱支持深色和浅色两种主题：

1. 点击左侧菜单最下方的「⚙ 设置」
2. 在「主题设置」区域，点击「深色主题」或「浅色主题」
3. 界面会立即切换，您的选择会被记住，下次打开自动应用

| 主题 | 特点 |
|------|------|
| 🌙 深色主题 | 黑色背景，护眼，适合晚上或光线暗的环境 |
| ☀️ 浅色主题 | 白色背景，清晰明亮，适合白天使用 |

## 常见问题

### Q：工具需要联网吗？

**A**：不需要！工具箱完全离线运行，所有处理都在您的电脑上完成，不会上传任何文件，保护您的隐私。

### Q：支持哪些图片格式？

**A**：支持常见的 JPG、PNG、WebP、BMP、TIFF、GIF 等格式。

### Q：处理大文件会很慢吗？

**A**：工具箱使用高效的处理方式，一般几秒内即可完成。文件越多、越大，处理时间会相应增加，但会有进度条显示处理状态。

### Q：不小心删除了文件能恢复吗？

**A**：使用「文件去重」功能删除的文件无法直接恢复。删除前请务必确认，建议先备份重要文件。

### Q：输出的文件保存在哪里？

**A**：每个工具都可以选择输出目录。如果不选择，一般默认保存到原文件所在目录，文件名会加上「_compressed」、「_scaled」等后缀以示区别。

### Q：可以同时打开多个工具吗？

**A**：可以！左侧菜单点击不同工具会切换工作区域。您可以在一个工具处理完成后，切换到另一个工具继续使用。

### Q：深色主题和浅色主题有什么区别？

**A**：只是外观颜色不同。深色主题背景偏黑，适合暗光环境；浅色主题背景偏白，适合明亮环境。功能完全一样，按个人喜好选择即可。

### Q：为什么有些按钮是灰色的不可用？

**A**：通常是因为还没添加文件，或者还没选择必要的选项（如输出路径）。添加文件并填写必要信息后，按钮会自动变为可用状态。
